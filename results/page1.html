What Is Machine Learning?

Machine learning is the science (and art) of programming computers so they can learn fr om data.

Here is a slightly more general definition:

[Machine learning is the] field of study that gives computers the ability to learn without being explicitly pr ogrammed.

-Arthur Samuel, 1959

And a more engineering-oriented one:

A computer pr ogram is said to learn fr om experience E with r espect to some task T and some performance measur e P, if its performance on T, as measur ed by P, impr oves with experience E.

-Tom Mitchell, 1997

Your spam filter is a machine learning program that, given examples of spam emails (flagged by users) and examples of regular emails (nonspam, also called “ham”), can learn to flag spam. The examples that the system uses to learn are called the training set.

- Machine learning involves programming computers to learn from data, allowing them to adapt without explicit programming.

- Machine learning encompasses various subfields, including supervised and unsupervised learning, online and batch learning, and instance-based and model-based learning.

- The workflow of a machine learning project typically involves data collection, model selection, training, evaluation, and fine-tuning.
